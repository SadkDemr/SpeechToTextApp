Adım-1 Kütüphanler

Projede python web framework içeriği olan Flask kullanılmıştır. Gelen sesi yazıya çevirmek içinde speech_recognition kütüphesi kullanılmıştır.

from flask import Flask, render_template, request, redirect
import speech_recognition as sr

Adım-2 Çeviri

Bu kısım ise yüklenen ses dosyasının metine dönüştürülüp yazıdırılasıdır.

app = Flask(_name_)
@app.route("/", methods=["GET", "POST"])
def index():
transcript = ""
if request.method == "POST":
print("FORM DATA RECEIVED")
if "file" not in request.files:
return redirect(request.url)
file = request.files["file"]
if file.filename == "":
return redirect(request.url)
if file:
recognizer = sr.Recognizer()
audioFile = sr.AudioFile(file)
with audioFile as source: data = recognizer.record(source)
transcript = recognizer.recognize_google(data, key=None)
return render_template('index.html', transcript=transcript)
